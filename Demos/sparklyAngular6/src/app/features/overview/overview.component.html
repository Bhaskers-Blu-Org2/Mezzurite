Welcome to Sparkly Web

This is a demo site to show how to do JSLL telemetry for a SPA page

This site was built to show case JSLL, and Mezzurite in an Angualr application. With JSLL we are targeting two concepts, the use of a cV to track impressions, and a way to track the first call from a browers to a CDN/Front Door.

The chart on the left is showing the flow of JSLL events fire by this site. The chart on the right is showing WC3 Navigation Timings overlayed with Mezzurite
